1. Download Visual Studio and install the C++ language files.
2. Download Diannex from the GitHub repository and compile it by using Visual Studio.
3. Change the "main.dx" file as much as you want.
4. Compile the "UnderStudio Core Engine.json" file by using the Command Line your operating system has.
5. To compile, type in "diannex --project "[Path to the engine file you've installed]\Diannex Example Project\UnderStudio Core Engine\UnderStudio Core Engine.json""
